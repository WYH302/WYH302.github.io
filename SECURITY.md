# Public Website Safety Notes

This repository is intended to become a public GitHub Pages site. Treat every
file committed here as public.

## Do Not Commit

- API keys, tokens, passwords, cookies, or private SSH keys
- Unpublished paper PDFs unless all authors and advisors approve public release
- Raw datasets with privacy, ethics, copyright, or license restrictions
- Recommendation letters or private admissions documents
- Passport, ID card, student card, transcript, or visa scans
- Private reviewer comments, collaborator notes, or internal lab materials
- Personal phone number, home address, or private calendar links

## Before Publishing

- Replace placeholder content with accurate academic information
- Confirm that the CV PDF is intended for public distribution
- Check that every linked paper, poster, dataset, and code repository can be
  shared publicly
- Use a university email address if possible
- Keep third-party images, figures, and tables out of the site unless you have
  permission or a clear license

## If Sensitive Data Is Exposed

1. Remove the file from the repository immediately.
2. Rotate any exposed token, password, or API key.
3. Ask GitHub Support about purging cached Pages artifacts if needed.
4. Review git history for similar exposures before publishing again.
