This page is ready for two or three verified research projects. It intentionally avoids displaying invented project titles, fake datasets, or unapproved research outputs.

How to add a project

Use a problem-method-result-contribution structure

  • Problem: State the specific research question.
  • Method: Describe the model, dataset, experiment, proof, or analysis.
  • Result: Report the most important evidence without exaggeration.
  • Contribution: Clarify what was personally designed, implemented, analyzed, or written.

Public links

Link only materials that are approved for sharing

Code repositories, posters, reports, slides, and papers should be public only when all collaborators, advisors, copyright terms, and data restrictions allow publication.
