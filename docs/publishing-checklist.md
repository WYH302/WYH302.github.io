# Publishing Checklist

Use this checklist before making the site public.

## Identity

- [ ] Replace `[Your Name]` everywhere
- [ ] Replace department, university, city, and field placeholders
- [ ] Replace advisor placeholder or remove it if not appropriate
- [ ] Replace the profile placeholder image with a formal photo
- [ ] Replace `your_email@university.edu`
- [ ] Replace `yourusername` and all profile links

## Academic Content

- [ ] Research interests are specific and not generic
- [ ] Projects explain problem, method, result, and personal contribution
- [ ] Publications are labeled accurately
- [ ] Working papers are not presented as accepted papers
- [ ] Blog posts are research notes, not casual social updates
- [ ] CV is current and named `assets/files/cv.pdf`

## Safety

- [ ] No secrets, tokens, credentials, or private config files
- [ ] No restricted papers, data, review comments, or internal materials
- [ ] No ID documents, transcripts, recommendation letters, or private records
- [ ] Third-party images and figures have permission or clear attribution

## Technical

- [ ] `npm run check` passes
- [ ] Local preview works at `http://localhost:8080`
- [ ] Navigation works on desktop and mobile
- [ ] GitHub Pages build completes
- [ ] `https://<yourusername>.github.io` loads over HTTPS
